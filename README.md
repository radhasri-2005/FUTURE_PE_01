# AI Cafe Website Copy Project

## Business
CafeAura  
Location: Visakhapatnam

## Objective
To design a structured AI prompt system that generates high-quality, conversion-focused website copy for a cafe.

## Problem
Many local businesses lose customers due to:
- Unclear website content
- Weak value proposition
- Poor calls-to-action
- Generic or outdated website content

## Solution
This project uses prompt engineering to generate:
- Homepage content
- Menu descriptions
- Booking system content
- CTA sections

## Tools Used
- ChatGPT (for prompt design)
- Claude AI (for generating website content)

## Features
- Clear and engaging content
- Strong call-to-actions (CTAs)
- Booking system included
- Customer-focused messaging
- Ready-to-use website content

## Prompt Logic
The system takes business details (name, location, services, audience) and generates structured website content that is:
- Conversion-focused
- Easy to understand
- Adaptable for different businesses


## Outcome
This project demonstrates how AI can help local businesses improve their online presence, attract more customers, and increase bookings.

## Use Case
Can be used by:
- Cafe owners
- Freelancers
- Web agencies

## Key Benefits
- Improved clarity
- Better customer engagement
- Higher conversion potential
